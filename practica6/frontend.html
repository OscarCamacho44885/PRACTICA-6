FRONTEND

El frontend o también conocido como "desarrollo del lado del cliente" se refiere que es la práctica de producir HTML, CSS y JavaScript, lo cuales estos se encargan de dar forma a la parte frontal de un sitio web o aplicación, como fondos, colores, texto, animaciones o efectos

¿Para que sirve el Frontend?

El frontend sirve para realizar la interfaz de un sitio web, desde su estructura hasta los estilos, como pueden ser la definición de los colores, texturas, tipografías, secciones, entre otros. Su uso es determiante para que el usuario tenga buena experencia dentro del sitio o aplicación
